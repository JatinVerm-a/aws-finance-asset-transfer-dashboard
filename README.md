# AWS Finance – Global Asset Transfer Dashboard

## 📊 Project Overview
This project simulates AWS Finance Operations using Excel. It focuses on tracking global asset transfers, vendor performance, delay analysis, and cost optimization.

## 🎯 Key Features
- KPI tracking (Total Transfers, Delay Rate, Avg Transfer Time, Total Cost)
- Vendor performance analysis
- Delay reason root cause analysis
- Interactive dashboard with slicers
- Cost optimization insights

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Analysis

## 📈 Insights
- Identified high delay rate (57%) indicating operational inefficiencies
- Vendor delays were the major contributing factor
- Cost differences across vendors highlighted optimization opportunities
